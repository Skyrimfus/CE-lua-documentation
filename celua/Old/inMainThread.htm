Index 
inMainThread

Check if the current code is running inside the main thread

inMainThread( )


Return Value

Returns true if the current code is running inside the main thread, otherwise returns false




Example




Copy text

function print_if_main_thread()
if inMainThread() then print("INSIDE MAIN THREAD") else print("NOT INSIDE MAIN THREAD") end
end

print_if_main_thread()--OUTPUT: INSIDE MAIN THREAD

createThread(print_if_main_thread)--OUTPUT: NOT INSIDE MAIN THREAD
