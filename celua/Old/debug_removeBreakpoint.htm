Index 
debug_removeBreakpoint

Removes a breakpoint from an address

debug_removeBreakpoint( address )


Parameters




addressAddress of the breakpoint




Return Value

Returns true if the breakpoint has been removed, otherwise return false



Remarks

It will return false if there was no breakpoint at the address
