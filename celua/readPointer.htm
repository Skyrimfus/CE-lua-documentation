readPointer Index 
readPointer

Read a 8-byte/4-byte value depending on the process bitness

readPointer( address )


Parameters
addressThe address of the value to be read
Return Value

In 64-bit processes: Returns the 8-byte value at the specified address
In 32-bit processes: Returns the 4-byte value at the specified address



Remarks

If the memory can't be read, the return value is nil

The address is typically an integer, but it can be represented as a string(AddressString) aswell
