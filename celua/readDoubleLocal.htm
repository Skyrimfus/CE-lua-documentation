readDoubleLocal Index 
readDoubleLocal

Read a 8-byte, double-precision, floating point value (inside Cheat Engine's memory)

readDoubleLocal( address )


Parameters
addressThe address of the value to be read
Return Value

Returns the double-precision, floating point value at the specified address inside Cheat Engine


Remarks

Performs the read operation inside the Cheat Engine process itself, rather than the attached process
If the memory can't be read, the return value is nil

The address is typically an integer, but it can be represented as a string(AddressString) aswell
