isKeyPressed Index 
isKeyPressed

Check if the given key is pressed

isKeyPressed( key )


Parameters
keyKey to check for
Return Value

Returns true if the key is pressed, otherwise returns false



Remarks

The key can either be a keycode or a string char(examples: "a", "b", "f")
CE Lua already has some keycodes defined as global variables(VK_ESCAPE, VK_CONTROL, VK_A, VK_B, ...)
