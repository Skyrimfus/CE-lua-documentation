readByte / readShortInteger Index 
readByte / readShortInteger

Read a single byte

readByte( address [, bSigned=false])


readShortInteger( address [, bSigned=false])


Parameters
addressThe address of the byte to be read
bSigned[optional] If true the value returned will be signed
default(false)
Return Value

Returns the byte at the specified address



Remarks

If the memory can't be read, the return value is nil

The address is typically an integer, but it can be represented as a string(AddressString) aswell

readShortInteger is the alias for readByte
