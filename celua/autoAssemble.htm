autoAssemble

Execute a Auto Assemble script

autoAssemble( script [, tblDisableInfo = nil])


autoAssemble( script [, bTargetSelf = false [, tblDisableInfo = nil ]])


Parameters
scriptAuto Assemble script that will execute
tblDisableInfo[optional] The table that autoAssemble returns. If this parameter is passed then the [DISABLE] section of the script will be executed
default(nil)
bTargetSelf[optional] If true, then the script will be assembled in Cheat Engine's memory, rather than the attached process'
default(false)
Return Value

2 return values:
If the script executes sucessfully then:
  The first return value is true
  The second return value is a table that you can use in the tblDisableInfo parameter

If the script fails to execute then:
  The first return value is false
  The second return value is a string that tells you why the script failed
