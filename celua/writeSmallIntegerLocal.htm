writeSmallInteger Index 
writeSmallInteger

Write a word(2-byte) value (inside Cheat Engine's memory)

writeSmallInteger( address, value )


Parameters
addressThe address where the word(2-byte) value will be written
valueThe word(2-byte) value to write
Return Value

Returns true if the write was successful
If the write fails, the return value is false


Remarks

Performs the write operation inside the Cheat Engine process itself, rather than the attached process

The address is typically an integer, but it can be represented as a string(AddressString) aswell
