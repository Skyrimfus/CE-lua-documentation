readInteger

Read a dword(4-byte) value

readInteger( address [, bSigned = false] )


Parameters
addressThe address of the value to be read
bSigned[optional]If true the value returned will be signed
Return Value

Returns the dword(4 bytes) at the specified address



Remarks

If the memory can't be read, the return value is nil

The address is typically an integer, but it can be represented as a string(AddressString) aswell



Related

readByte, readBytes, readSmallInteger, readQword, readFloat, readDouble, readPointer, readString
writeBytes, writeSmallInteger, writeInteger, writeQword, writeFloat, writeDouble, writePointer, writeString
