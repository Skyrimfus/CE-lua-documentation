printf

Prints formated text to the output in Lua Engine window

printf( formatedString [, variable1 [, variableN ,[ ... ]]] )


Parameters




formatedStringA string to be formated


variableNVariable(s) that will be used in formatting




Remarks

Same as print(string.format(formatedString [, variable1 [, variableN ,[ ... ]]]))



Related

print



Example



printf("0x%X",15)
--outputs "0xF"
