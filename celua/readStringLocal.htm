Index 
readString

Reads a string till it encounters a 0-terminator (In Cheat Engine's memory)

readStringLocal( address [, maxlength [, bWideChar]] )


Parameters




addressThe address of the string to be read


maxlengthMax length of the string in bytes


bWideCharSet this to true if you want to read the string as a wide char




Return Value

Returns the string at the specified address



Remarks

Maxlength is just so you won't freeze for too long, set to 6000 if you don't care too much



Example
readString(address, maxlength, widechar OPTIONAL) : Reads a string till it encounters a 0-terminator. Maxlength is just so you won't freeze for too long, set to 6000 if you don't care too much. Set WideChar to true if it is encoded using a widechar formatting
