print

Prints text to the output in Lua Engine window

print( text [, text2 [, ... ]] )


Parameters
textText to be printed
Return Value

This function returns the outputted string



Remarks

Print can only output numbers and strings. If you want to output booleans, nil values or other types use printf or manually convert them to a string using tostring()
