writeString

Writes a string (inside Cheat Engine's memory)

writeString( address, string [, bWideChar=false] )


Parameters
addressThe start address where the string will be written
stringThe string to write
bWideChar[optional] Set this to true if you want to write a wide-string
default(false)
Return Value

Returns true if the write was successful
Returns false if the write fails



Remarks

Performs the write operation inside the Cheat Engine process itself, rather than the attached process

The address is typically an integer, but it can be represented as a string(AddressString) aswell

This function does not append a 0-terminator at the end of the string
