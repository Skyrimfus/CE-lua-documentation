readSmallIntegerLocal

Read a 2-byte value (inside Cheat Engine's memory)

readSmallIntegerLocal( address [, bSigned=false])


Parameters
addressThe address of the byte to be read
bSigned[optional] If true the value returned will be signed
default(false)
Return Value

Returns the word(2 bytes) at the specified address inside Cheat Engine



Remarks

Performs the read operation inside the Cheat Engine process itself, rather than the attached process
If the memory can't be read, the return value is nil

The address is typically an integer, but it can be represented as a string(AddressString) aswell
