Index 
debug_getXMMPointer

Short description

debug_getXMMPointer( xmmRegisterNumber )


Parameters




xmmRegisterNumberNumber of the XMM register (e.g. for XMM0 it's 0, for XMM3 it's 3...)





Return Value

  Returns the address(local CE)  of the specified xmm register of the thread that is currently broken
  
  

Remarks

  This is a LOCAL Cheat Engine address. Use Local memory access functions to read and modify
  xmmRegisterNumber can be 0 to 15 (0 to 7 on 32-bit)
