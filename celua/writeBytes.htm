writeBytes

Write multiple bytes

writeBytes( address, byte [, byte2 [, byte3 ,[...]]] )

Write a table of bytes

writeBytes( address, table )


Parameters
addressThe start address of the byte(s) to be written
byteValue of the byte to be written
tableA table containing the bytes to be written
Return Value

Returns the ammount of bytes written



Remarks

The address is typically an integer, but it can be represented as a string(AddressString) aswell
The table should be a standard lua table, where the first byte is at index 1 and the last byte is at index #table(length of table)
