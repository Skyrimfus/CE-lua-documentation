Index 
createTimer

Create a timer object(TTimer) that executes a function periodically:

createTimer( [parent = nil [, bEnabled = false]] )

Create a timer object(TRunOnceTimer) that waits a given delay, executes a function and selfdestroys:

createTimer( delay, function )


Parameters

TTimer:



parent[optional] Parent object


enabled[optional] Start the timer enabled



TRunOnceTimer:



delayAmmount of time to delay the execution of the function(in milliseconds)


functionFunction to execute




Return Value

Returns the created timer object



Related

TTimer Object



Example



if tm then tm.destroy() end -- if tm is a value other than nil, that means the timer already exists so, destroy it
tm = createTimer()
tm.Interval = 1000--1 second
tm.OnTimer = function()
  print("Tick")
end

createTimer(10000,function()--After 10 seconds destroy the original timer
  tm.destroy()
  print("Destroying the Tick timer")
end)
