Index 
do...end

Create a code block

do
statements
...
end


Remarks

Local variables created inside the block will get destroyed after the end statement

Statements such as loops:while, for, conditionals:if, and function declarations: function, all use do...end as their bodies


Related

for...do, for...in...do, while, if...then



Example



var = 5
do
  var=var+10
  local local_var = 9
  print("Local var inside do..end:",local_var)--outputs 9
end
print("Local var outside do..end:",local_var)--outputs nil because the variable was local to the do...end scope, therefore it gets destroyed when we leave that scope
print("Global var outside do..end:",var)--outputs 15
