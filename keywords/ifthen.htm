Index 
if...then...end

Conditionally run statements

if <expression> then statement end


Parameters


expressionIf the expression is true, the statement is executed.


Remarks

The expression can contain the boolean operators of and, or, and not as well as the logical operators <, <=, >, >=, ==, and ~= grouped with parentheses as needed.

In Lua, any value may represent a expression. Lua consider false and nil as false and anything else as true: Beware that, unlike some other programming
languages, Lua considers both zero and the empty string as true in conditional statements. 


Related

if..else..elseif..end


Example



num = 2
if num%2==0 then print("Even number") end
