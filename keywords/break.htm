Index 
break

Stops the execution of the loop

break


Remarks

After using break, the program will continue executing the code after the loop end
You can use break to escape for, while and repeat loops


Related

for...do, for...in...do, while, repeat



Example: Infinite loop



var = 0
while true do--infinite loop
  var = var+1 --increase var by 1 each loop
  if var > 10 then break end--infinite loop will stop when var is greater than 10
end
