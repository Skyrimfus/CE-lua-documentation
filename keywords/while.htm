Index 
while

Loop based on an expression.

while <expression> do
statements
...
end



Parameters




expressionIf the expression is true the statements up to end will be executed. This loop continues until the expression is false.




Remarks

The expression is tested before the loop is executed so the loop will be executed zero or more times.
To create an infinite loop, you can use a true constant as the expression.



Related

break, for...in...do, repeat...until



Example



var = 0
while true do--infinite loop
  var = var+1 --increase var by 1 each loop
  if var > 10 then break end--infinite loop will stop when var is greater than 10
end
